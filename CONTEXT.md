# Alfredo Landing Page

The marketing page for Alfredo: one dashboard for every self-hosted project. This glossary holds the product's coined vocabulary. The coined set deliberately stops at three terms; everything else on the page is plain English. Page copy and demo pixels stay plain too — coined nouns appear on the page only inside the glossary section (wayfinder #34).

## Language

**HQ**:
The single dashboard every project reports to: traffic, email, payments, errors, uptime. Long-form in prose is headquarters; counted or etched it is always HQ.
_Avoid_: console, control room, one box

**Porter**:
Alfredo's worker on one of your servers. Exactly one porter per server, the HQ's server included; it deploys projects, sets out trays, and reports back to the HQ.
_Avoid_: worker, agent, daemon

**Tray**:
One integration, ready to use: analytics, email, payments, auth. A tray is set out once and is project-independent; every project that selects it is served from the same tray. Covers both self-hosted services and wired external accounts.
_Avoid_: per-project connection

### Retired

- **fleet** — retired 2026-07-16 (wayfinder #34); use plain English ("projects", "all your projects"), including in demo etches.
